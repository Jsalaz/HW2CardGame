Remote Github Push Pull Test for CSC496
1) Download the Repository: 
	git clone https://github.com/Jsalaz/HW2CardGame.git
2) Open the README.txt
3) Add your name to the end of the file
4) Add the change to the repository:
	git add README.txt
5) Commit the change:
	git commit
6) Push the change to the online repository:
	git push origin master
7) Check the git log:
	git log
8) Pull most recent update from online repository. (wait a day or so)
	git pull
